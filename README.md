# ICML 2026 中文日程表

这是一个单页静态 HTML 网站，可以直接部署到 GitHub Pages。

## 部署方式

1. 在 GitHub 新建一个公开仓库，例如 `icml-2026-schedule`。
2. 上传本目录中的 `index.html`、`.nojekyll`、`README.md`。
3. 进入仓库 `Settings` → `Pages`。
4. 在 `Build and deployment` 中选择 `Deploy from a branch`。
5. Branch 选择 `main`，Folder 选择 `/(root)`，点击 `Save`。
6. 稍等几分钟，访问：`https://<你的GitHub用户名>.github.io/icml-2026-schedule/`。

## 更新

修改 `index.html` 后提交到 `main` 分支，GitHub Pages 会自动重新部署。

## 说明

日程基于 ICML 官方公开页面和公开日程信息整理，具体房间与临时变动请以 ICML 官方 App / 官方 Calendar 为准。
